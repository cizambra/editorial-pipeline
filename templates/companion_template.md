🚨 FOR AI: CRITICAL INSTRUCTIONS

Read this entire section BEFORE writing. 
Every rule matters.

Rule 1: 
VOICE TRAINING (DO THIS FIRST)
Before writing a single word:
1. Write TO the reader (second person: "you"), NOT ABOUT them
2. Be direct and concrete - NO vague, poetic language.
3. The tone must be dinner table talk. Talk that the reader can understand as if you were sharing a meal together.

BANNED WEASEL WORDS (never use these):
* "quiet" (heaviness, noise, etc.)
* "honest" (step, move, etc.)
* "clean" (rep, close, etc.)
* "gentle" (practice, return, etc.)
* "deep" (work, dive, etc.)
* Anything vague or empty. Expressions that add humanity like actually, just, really are okay as long as it's sporadic.

Instead use:
* ✅ Concrete nouns: "two minutes," "three breaths," "one sentence"
* ✅ Specific verbs: "notice," "write," "run," "log"
* ✅ Real stakes: "energy," "time," "capacity"

Rule 2: 

KATA SELECTION & NAMING - PREFER ESTABLISHED PRACTICES

STEP 1: PRIORITIZE ESTABLISHED PRACTICES

When selecting Katas for a companion, FIRST look for established practices that fit the struggle.


Why this matters:
* Built on validated research/methods
* Readers may already be familiar (builds trust)
* Shows intellectual honesty (integration, not invention)
* Avoids reinventing the wheel
* Gives proper attribution

Examples of established practices to consider:

* Regulation: The Physiological Sigh (Huberman), Box Breathing (Navy SEALs), 4-7-8 Breathing (Weil)
* Grounding: 5-4-3-2-1 Technique, Body Scan (MBSR), Progressive Muscle Relaxation
* Focus: Pomodoro Technique (Cirillo), Implementation Intentions (Gollwitzer)
* Decision-making: Eisenhower Matrix, Two-Minute Rule (Allen)
* Closure: Zeigarnik Effect applications, Shutdown Ritual (Newport)

STEP 2: NAMING ESTABLISHED PRACTICES
If using an established practice, USE ITS PUBLIC NAME and cite the source:

#### 1️⃣ The Physiological Sigh
*Based on research by Dr. Andrew Huberman (Stanford)*

This kata trains regulation on demand. Use it when your thoughts race...

[Rest of kata structure...]

STEP 3: WHEN TO CREATE ORIGINAL KATAS
Only create new Katas when:
* No established practice fits the specific capacity you're training
* You're combining multiple techniques into one new practice
* You're adapting an established method significantly for neurodivergent use

If creating an original Kata:
* Name it clearly and descriptively
* Make it reusable across contexts
* Keep it 30-120 seconds
* Make it so its practicable before needing it

If creating an original Kata, do NOT add any disclaimer or note explaining why no established practice was used. Just write the kata. The absence of a source citation is sufficient to signal it is original.

WRONG (reinventing established practice):
#### 1️⃣ The Calm Reset
Breathe in for 4, hold for 7, breathe out for 8...

This is just 4-7-8 Breathing (Dr. Andrew Weil). Use the established name.

RIGHT (using established practice):
#### 1️⃣ 4-7-8 Breathing
*Developed by Dr. Andrew Weil*

This kata trains nervous system down-regulation...

Rule 3: REFERENCE THE FREE REFLECTION CORRECTLY

WRONG: "The reflection you read named that feeling..." 
RIGHT: "In our latest reflection ([link to reflection]), we named that feeling..."

Structure:
In our latest reflection ([Reflection Title](link-to-reflection)), we [what was covered].

Rule 4: VARY PHYSICAL CUES ACROSS COMPANIONS
Match the physiology to the context. Don't repeat the same pattern.
